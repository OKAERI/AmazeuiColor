Radio 单选框

单选框和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。Amaze UI Color 兼容webkit浏览器，Chrome、Safari以及Opera浏览器生效。

Radio 单选框 列表
使用方法

在 input 添加 checked 更改默认。

Radio 单选框 颜色
使用方法

在 .color-radio 的父级元素添加 .color-error 、 .color-success 、 .color-warning 、 .color-info 即可改变单选框颜色。
