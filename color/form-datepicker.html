DatePicker 日期选择框

输入或选择日期的控件。此组件基本使用部分，是基于Amaze UI 时间框 ，更多功能以及API请移步 Amaze UI 官方手册。

DatePicker 基本使用
使用方法

在 input 上增加 data-am-datepicker属性，调用日期插件。

API说明
参数 说明 类型 默认值
trigger 目标输入框ID。 String 必填
minDate 最小日期 注意：该值会覆盖标签内定义的日期范围 String 必填
maxDate 最大日期 注意：该值会覆盖标签内定义的日期范围 String 必填
type 调出日期选择 date 默认 time 调出时间选择 ym 调出年月选择 String 必填
