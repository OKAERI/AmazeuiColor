使用方法
API说明
参数 说明 类型 默认值
max 最大长度,如果不传会去取文本框的maxlength Number 必填
isOverflowCut 是否自动截取文本 Boole true false
overClass 显示计数的结点 String
withButton 关联按钮 String
minHeight 文本框的最小高度，因为这里做自适应高度的控制。如没有此参数时，不自适应高度。 Number -
overflowCallback 超出时的回调，this指向当前对象,n为长度,textbox是文本框结点对象,max为最大长度
changeCallback 长度改变时的回调,n为长度,textbox是文本框结点对象,max为最大长度
passClallback 长度通过时的回调,n为长度,textbox是文本框结点对象,max为最大长度
isByte 是否按字节数来计算，true时：一个汉字作2个单位长度，false时汉字英文不作区分 Boole true false
