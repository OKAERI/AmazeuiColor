Amaze UI Color

Amaze UI Color 是基于 Amaze UI 开发而来的web组件集，主要针对移动端体验更加友好。

获取源码

Amaze UI Color 1.0.1

使用方法

Amaze UI Color 组件集依赖于 Amaze UI ,所以在引入Color 组件集文件时，必须预先引入 Amaze UI 的 css、js 文件。

当前 v1.0.0 说明

1.0.0 版本主要更新了form表单类组件，针对Amaze UI 缺失的组件部分进行了填充，并对大部分已有组件进行二次美化，针对移动端进行了更进一步优化，有了更好的体验。

预览 v1.1.0 说明

讲针对CSS基础组件开始填充，辅助类、按钮、输入框、步骤进度，优化、美化以及填充。

  1. 辅助类：颜色、字体等
  2. 按钮：图标按钮、状态按钮、组合按钮
  3. 输入框：提供更多样式的输入框
  4. 步骤进度条：切换下一步、横向进度条、竖向进度条
  5. 标签输入条：属性标签
